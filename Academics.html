Capstone Projects

Explore the impactful capstone projects undertaken by our students.

Project Overview:

The SecureChat Application is a comprehensive messaging platform designed with a focus on user privacy and data security. The goal of this capstone project is to develop a secure, end-to-end encrypted messaging application that ensures the confidentiality and integrity of user communication.

Key Features:

  • End-to-End Encryption:
    • Implement robust encryption algorithms to secure messages during transmission.
    • Utilize public-key cryptography to enable secure key exchange between users.
  • User Authentication:
    • Develop a secure authentication system to verify the identity of users.
    • Implement multi-factor authentication for an additional layer of security.
  • Secure File Sharing:
    • Enable users to securely share files through the application.
    • Implement encryption for stored files on both sender and recipient devices.
  • Anonymous Chat Rooms:
    • Introduce the concept of anonymous chat rooms for users who prefer heightened privacy.
    • Implement measures to prevent tracking and identification within these chat rooms.
  • User-Friendly Interface:
    • Design an intuitive and user-friendly interface for seamless user interaction.
    • Incorporate features such as message timestamps, read receipts, and customizable profiles.

Technologies Used:

  • Programming Languages:
    • Java for Android application development.
    • Swift for iOS application development.
    • Node.js for backend server development.
  • Encryption Libraries:
    • Use OpenSSL or Bouncy Castle for implementing encryption algorithms.

Expected Outcomes:

  • A fully functional, cross-platform messaging application.
  • Detailed documentation on the application's architecture, security features, and usage.
  • Presentation and demonstration of the application's security features.
  • Recommendations for future enhancements and potential real-world applications.
